What is Jabber?

Jabber is a federated chat network.

What separates it from other chat networks like Skype or MSN is the fact that it's made up of a lot of individually run servers that talk between themselves, similar to how Email works.

Jabbit.me is just one of these servers, and someone with an account here (username@jabbit.me) can talk to anyone on the network, even if they're registered on a different server entirely (such as jabber.org).

Why would I use it?

There are many reasons to use Jabber. Some of the most notable one being:

Security and Privacy

Most Jabber servers are both highly secure and open source. In fact, you can check the transport security of any given server yourself on XMPP.net.

And, if you're really concerned about your messages being secret, you can always use client-side encryption (OTR) or even set up your own server.

Clients and compatibility

Jabber uses a simple, extensible protocol called XMPP, supported by quite a few clients, covering just about every platform there is.

It's also what powers Facebook Chat and Google Talk, so Jabber clients can generally connect to those two as well.

Impossible to take down

  • Jabber has no central nodes that can fall.
  • It's not owned by a company that can shut it down on a whim.
  • No government can demand it to be shut down.

As long as there is at least one person in the world willing to run a server, Jabber isn't going away.

How do I use it?

To use Jabber, you only need three things: A client, a server, and an account.

The Client

  • Instantbird (Windows, Mac and Linux)
  • Pidgin (Windows and Linux)
  • Adium (Mac)
  • IM+ (iOS, Android, BlackBerry, Windows Phone)

For new users, we recommend one of the above clients. There are plenty of others to choose from, but these are the ones we believe will suit most people's tastes.

Pidgin and Adium are virtually identical under the hood, they're just different frontends for the same thing.
There is also a plugin for Pidgin and Adium that allows you to route Skype through it.

The Server

There are plenty of servers around to choose from.
If you want an account on jabbit.me (your-username@jabbit.me), you can register either using your client (see below)

For other candidates, we recommend XMPP.net's list.
Note: some of them have issues, like randomly disconnecting or not delivering messages. You'll just have to try and see.

The physical location of your server doesn't matter much, so you can usually choose the one whose name you like the best.
Also try not to pick one with an "F" security rating.

The Account

The main way to register for an account on an XMPP Server is using in-band registration. This can be done with either Pidgin or Adium (and plenty of other clients).

Note that not all servers support this - some require you to register on their website, or even message an admin and ask nicely.

Using Pidgin

  • If this is your first time launching Pidgin, the Add Account wizard will be opened for you. Otherwise, open it yourself with:
    Accounts → Manage Accounts → Add.
  • Enter these values:
    Protocol:
    XMPP
    Username:
    anything you like
    Domain:
    jabbit.me (or your domain of choice)
    Resource:
    name this computer something
    Password:
    anything you like
  • Check the checkbox titled Create this account on the server and press Save.

Using Adium

  • Open the Accounts settings with Adium → Preferences, then click Add → XMPP (Jabber).
  • Click Register New Account
  • Choose a domain to register on (such as jabbit.me) and press Request New Account
  • Fill in your desired username and password.
    If the server supports in-band registration, pressing OK should tell you that the registration succeeded, otherwise it will give you an error describing what went wrong.
  • Enter your newly registered details on the login page and press OK.
